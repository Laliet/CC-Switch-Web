# CC Switch v0.9.1

> `0.9.x` 修正版本更新说明

---

## 📦 v0.9.1 更新内容（2026-03-11）

这次更新是一次面向 `0.9.x` 版本线的**问题修复发布**，重点处理发布后暴露出的配置生效与兼容性问题。

`v0.8.0` 仍然是当前推荐稳定版本；如果你已经在验证 `0.9.x`，建议升级到 `v0.9.1`。

### ✅ 重点修复

- **修复当前生效 provider 保存/应用未立即写入 live 配置**
  - 现在编辑并保存当前正在使用的 provider 后，会立刻同步到实际配置文件
- **修复 Codex 当前 provider 更新时 MCP 配置被覆盖**
  - 例如 `relay-pulse` 这类已启用 MCP 条目，不会再因为更新当前 provider 而丢失
- **兼容 relay-pulse 当前健康检查返回结构**
  - 已支持基于 `groups[].layers[]` 的新响应格式，不再只依赖旧的 `data[]`

### 🔧 体验调整

- **明确顶部应用切换器语义**
  - 顶部应用切换器只负责切换管理视图，不直接修改 live 配置
  - 真正写入 live 配置的动作仍然是“启用 provider”或“保存当前生效 provider”
- **降低启动期无关面板的加载噪音**
  - 设置、提示词、MCP、技能页改为按需挂载，减少调试期的前端噪音

### 📌 发布说明

- **推荐稳定版本**：`v0.8.0`
- **修正预览版本**：`v0.9.1`
- 如果你追求更保守的生产环境使用体验，建议继续使用 `v0.8.0`
- 如果你已经部署或验证 `0.9.x`，建议升级到 `v0.9.1`
